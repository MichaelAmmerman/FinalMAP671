McDonald's Locations in Contiguous US

Michael Ammerman

McDonald's location data was provided by Gavin Rehkemper data accessed October 2023. State population and state shape file was provided by US Census Bureau. North American shape file was extracted from The University of Texas at Austin.

Tools used in this project

  • PostGIS was used to store/query the national dataset.
  • Data is shown on Mapbox web map with geolocation services enabled. If the map got popular it would require a fee to use.
  • Microsoft Excel.
  • Microsoft Access.
  • Oracle MySQL.
